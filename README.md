# 羽球線上分組系統 Pro

線上羽球分組與場地管理系統，支援單打/雙打模式，公開或私人房間，並提供比賽記錄、計分與計時功能。
---

## 功能特色

- 🏸 **單打 / 雙打模式**：自動分配球員至場地
- 🌐 **公開 / 私人房間**：可選擇是否顯示在大廳
- ⏱ **計時器 & 計分**：自動統計每場比賽
- 📊 **勝負紀錄**：保留歷史對戰資料
- 🔄 **快速重排**：一鍵重新分配球員
- 👥 **球員管理**：新增/刪除球員、設定上場權限
- 🔐 **密碼保護**：管理操作可設定密碼
- 🎨 **自訂字體大小與視覺風格**
---

## 專案檔案

|檔案|說明|
|Code.gs| Google Apps Script 後端邏輯（房間資料管理、同步功能） |
|index.html| 前端頁面（首頁、大廳、房間、管理面板） |
---

## 部署說明（Google Apps Script）

1. 打開 [Google Apps Script](https://script.google.com/)。
2. 建立新專案。
3. 上傳檔案：
   -Code.gs→GAS腳本
   -index.html→HTML前端
4. 點擊 **部署 → 新增部署→網頁應用程式**。
5. 設定執行權限與可存取對象（例如「任何人」或「僅自己」）。
6. 完成後，即可透過網址使用線上功能。
